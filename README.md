de.fighter456.wcf.pageMenuPermission 
====================================

This product extends the page menu administration of the WoltLab Community Framework. It adds the ability to provide the page menu items with user group privileges, without having to manually intervene in the database.

Version notes
-------------

This product is designed exclusively for the [WoltLab Community Framework](https://github.com/WoltLab/WCF) in version 1.1.X.

The development of this product has been discontinued!
